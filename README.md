WeatherSense AI is an AI-powered weather intelligence platform that combines real-time weather forecasting, air quality analysis, explainable AI, and Google's Gemini to deliver personalized weather insights through an interactive conversational assistant.

## 🚀 Why WeatherSense AI?

Most weather applications only display weather information.

WeatherSense AI goes a step further by interpreting weather conditions through multiple AI-driven modules that help users make informed daily decisions.

By integrating rule-based AI, explainable AI techniques, and a Large Language Model (Google Gemini), the application transforms raw weather data into meaningful recommendations while maintaining transparency in its decision-making process.

## 📖 Project Overview

WeatherSense AI is a full-stack AI-powered weather application developed using Flask, OpenWeather APIs, Google Gemini, and Chart.js.

The system provides:

- 🌤️ Real-time weather monitoring
- 📅 Five-day weather forecasting
- 🌍 Air Quality Index (AQI) analysis
- 📊 Interactive weather analytics
- 🧠 Explainable AI-based weather risk assessment
- 🏃 Outdoor comfort scoring
- 🩺 Personalized health recommendations
- 🤖 Context-aware AI weather assistant powered by Gemini
- 📍 Geolocation-based weather detection

The project demonstrates modern backend development, REST API integration, prompt engineering, asynchronous frontend communication, and practical AI application development.
